# read-and-write-a-file

This project contains two simple Python programs based on file handling.
*Task 1: Read a File*
Reads a file named sample.txt
Prints the content line by line
Shows an error message if the file is missing
*Task 2: Write and Append to a File*
Takes user input and writes it to output.txt
Appends more text to the same file
Displays the final content
*Files Used*
sample.txt (input file)
output.txt (output file)
